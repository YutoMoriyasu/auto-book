# auto-book

## Set Up

`$ python3 -m venv venv`

`$ . venv/bin/activate`

`(venv)$ pip install flask`

## Development

`$ . venv/bin/activate`

`(venv)$ export FLASK_ENV=development`

`(venv)$ flask run`


Press ctrl + C to exit development

If you want to build the server again, restart `$ flask run`
